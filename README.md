## 🛠️ Tecnologías y estándares de desarrollo
Para este trabajo práctico, el grupo se rige por los siguientes estándares técnicos obligatorios:
- **Variables**: Uso exclusivo de `let` y `const` para garantizar un alcance de bloque seguro.
- **Funciones**: Implementación de **Arrow Functions** (Funciones Flecha) en toda la lógica del proyecto.
- **Modularización**: Uso de `import` y `export` para una arquitectura de código limpia y escalable.
- **Paradigma**: Programación funcional, priorizando la inmutabilidad y métodos de arreglos.
